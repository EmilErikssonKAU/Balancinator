# Balancinator

This project implements a two-wheeled self-balancing robot using an Arduino, an MPU6050 IMU, and an L298N motor driver. The system uses the MPU6050’s DMP (Digital Motion Processor) to estimate orientation (pitch), and a PID controller to continuously adjust motor output to keep the robot upright.
